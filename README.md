# Minecraft Clone

## Middleware

* Unity 2017.4.1f1

## Preparing

### UnityPackages

Please import the following packages:

* Standard Assets

### Block texture generation

1. Move `Assets/MinecraftClone/Textures` directory
2. Enter `rake` command

**NOTICE**: _This command requires ImageMagick!_
